# Ancestry
Ancestry is a web application for creating family tree.

#### Building
```sh
npm run rebuild:all
```

#### Running
```sh
npm run start
```

#### Build and run tests
```sh
npm run test
```

####  Programming language
```sh
JavaScript, HTML5, CSS
```
